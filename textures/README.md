# Textures Directory

Place terrain textures here. Supported format: PNG, JPG.

Expected textures:
- `grass.jpg` — grass terrain
- `road.jpg` — road/dirt path
- `wheat.jpg` — wheat fields
- `forest.jpg` — forest floor
- `mud.jpg` — muddy terrain
- `stone.jpg` — stone/wall texture

The game will use these textures for terrain tiles when available.
