# Unit roster - by nation and type

All 576 units defined in `rwm_lang.sue`, grouped by **nation** then **type-class**.
Full machine-readable table with every field: `units_roster.csv`.

Columns in each entry: **Name** (shortname) - HP, front armor P=pierce/E=explosive,
speed, crew. **1:** primary weapon, **2:** secondary, **3:** tertiary
(`weapon_id dmg <damage> rng <range> rld <reload>`).
